In diesem Projekt wurde ein Reinforcement-Learning-Agent (Q-Learning) implementiert, der in einer Grid-World-Spielumgebung lernt, eine Prinzessin zu retten und sicher ins Schloss zurückzubringen.

Der Agent lernt durch Belohnungen und Bestrafungen, Hindernisse zu umgehen, Gegner zu vermeiden und einen möglichst guten Weg zu finden.
